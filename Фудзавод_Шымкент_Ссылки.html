GitHub Pages — как получить ссылку-страницу

  1. github.com → New repository (имя, напр. fudzavod-shymkent, Public, Add README) → Create.
  2. Add file → Upload files → загрузить HTML (лучше переименовать в index.html) → Commit.
  3. Settings → Pages → Source: Deploy from a branch → main → /(root) → Save.
  4. Через 1–2 мин появится адрес вида https://<логин>.github.io/fudzavod-shymkent/
